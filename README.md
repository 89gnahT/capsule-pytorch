# capsule-pytorch

##  汽车行业用户观点主题及情感识别 pytorch实现

CapsNet based on Geoffrey Hinton's original paper 
[Dynamic Routing Between Capsules](https://arxiv.org/abs/1710.09829)

## Requirements

 - python 3.6+
 - pytorch 0.4.1+
 - gensim
 - tqdm
   
## Run

```bash
python main.py
```
Train and test dataset should be included in data folder
